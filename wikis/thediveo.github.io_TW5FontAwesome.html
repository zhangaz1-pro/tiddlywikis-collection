Font Awesome Free 5 for TiddlyWiki 5

Simply download the  TW5FontAwesome plugin and then drag and drop it into your TiddlyWiki in order to enjoy  Font Awesome Free 5.

There is no need to install the font(s) into your operating system in order to use it with TiddlyWiki 5. Instead, simply drop this plugin into your TiddlyWiki and that's it.

Our  plugin demo wiki even features an interactive offline icon cheatsheet...
